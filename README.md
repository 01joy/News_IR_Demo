#使用方法
1. 安装jieba分词组件，命令为easy_install jieba
2. 安装Flask Web框架，命令为pip install Flask
3. 进入web文件夹，运行main.py文件
4. 打开浏览器，访问http://127.0.0.1:5000/ 输入关键词开始测试
